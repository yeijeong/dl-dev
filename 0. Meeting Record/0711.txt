1. 현재 진행 상황
예정 : 뉴스 과거 데이터 디비로(따로따로) -> 뉴스자동화(자연어 처리) -> 자연어처리도와주기(긍부정찾아보기) -> 뉴스 군집화 이런것들 찾아보기
수연 : 틱데이터 자동화
유진 : 자연어 처리 모델링 하는중(둘을 같이 병행해야할 듯)
단비 : 데이터 합치고, 전처리하고 하는 중
윤영 : 주식 전처리는 끝(코스피 지수 -> 실시간으로 하려면 크롤링 해야함) / 이거 끝나면 딥러닝 강화학습으로 넘어가~
창민 : 자연어 처리 어떻게 해야하는 지 모르겟당(찾아보는데 잘 안나옴) -> 모델링이 힘들다면 자연어처리 부분 해주세용

2. 정해야할 것
- 뉴스 디비 저장 어떻게 할 것인가 : 키워드별로 올리기
- 틱데이터 넘겨지는건 잘 되는데 자동화
- 뉴스 자연어처리 모델링 결과 : 등락률이 나온다!
