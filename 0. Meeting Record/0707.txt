- 예정 : 강화학습 공부하기
- 수연 : 틱데이터 주식 실시간 DB로 넘기는거
- 창민 : 코버트 사용하기
- 윤영, 단비 : 주식전처리
- 유진 : 과거주식데이터 가져온거랑 현재 뉴스 다운받고 있는거 병합 -> 등락률 옆에다 추가 하려함 날짜가 안맞아서 전처리중
    원래 가져왔던거 5개만 가져옴(과거), 틱데이터도 5개만 가져오는중 

- *주식 실시간 : 유진(M), 창민(디비로 넘기기만 하면 끝)
- *과거데이터(뉴스) : 예정(M), 수연(완료 데이터 수집만 하면됨됨)
- 뉴스 실시간 : 수연, 예정(M)(데이터 수집 돌리기만 하면 됨)

- *자연어처리 : 창민(M), 유진
- *과거데이터(주식전처리) : 단비(M), 윤영(7/7 오늘 내로 끝내기 혹은 주말까지 끝내오기)

- 강화학습 : 단비(M), 수연, 윤영
- 딥러닝 : 창민, 윤영(M), 단비
- DB : 수연(M), 예정
- 서비스 구현(웹사이트? 구현) : 예정(M), 수연
